# ute_check_proyecto
Sistema web para el control de asistencias académicas mediante roles (Administrador, Prefecto, Docente y Alumno). Implementa autenticación con JWT, gestión de usuarios, horarios y asistencias, usando Spring Boot, API REST y base de datos relacional.
